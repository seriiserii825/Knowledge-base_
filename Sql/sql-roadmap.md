### Этап 1: Основные SQL-команды (CRUD операции)

- SELECT - выборка данных, простые запросы
- WHERE - фильтрация данных, условия
- INSERT - добавление записей
- UPDATE - обновление данных
- DELETE - удаление записей

Операторы сравнения и логические операторы (AND, OR, NOT, LIKE, IN, BETWEEN)

### Этап 2: Проектирование таблиц

- Создание таблиц (CREATE TABLE)
- Первичные ключи (PRIMARY KEY)
- Типы данных и их правильный выбор
- Ограничения (NOT NULL, UNIQUE, DEFAULT)
- Изменение структуры таблиц (ALTER TABLE)

### Этап 3: Связи между таблицами

- Внешние ключи (FOREIGN KEY)
- Типы связей: один-к-одному, один-ко-многим, многие-ко-многим
- JOIN операции (INNER JOIN, LEFT JOIN, RIGHT JOIN)
- Нормализация баз данных (базовое понимание)

### Этап 4: Продвинутые запросы

- Агрегатные функции (COUNT, SUM, AVG, MIN, MAX)
- GROUP BY и HAVING
- ORDER BY и сортировка
- LIMIT и пагинация
- Подзапросы (subqueries)

### Этап 5: Индексы и оптимизация

Что такое индексы и зачем они нужны

- Создание индексов
- EXPLAIN - анализ запросов
- Базовая оптимизация запросов

### Этап 6: Дополнительные возможности

- Представления (VIEWS)
- Транзакции и ACID
- Хранимые процедуры (stored procedures) - базовое знакомство
- Триггеры - базовое понимание
- Пользователи и права доступа

### Этап 7: Практика

Создание реального учебного проекта (например, интернет-магазин, библиотека, блог)
Резервное копирование и восстановление данных
Работа с MySQL через язык программирования (PHP, Python, Node.js и т.д.)
