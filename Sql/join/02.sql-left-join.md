## LEFT JOIN (LEFT OUTER JOIN)

LEFT JOIN возвращает все строки из левой таблицы и совпадающие строки из правой таблицы.
Если совпадений нет, возвращаются NULL для столбцов правой таблицы.

### Синтаксис

```sql
SELECT columns
FROM table1
LEFT JOIN table2
ON table1.column = table2.column;
```

### Примеры запросов

**Пример 1: Простой LEFT JOIN**

```sql
SELECT
    c.name,
    c.city,
    o.product,
    o.amount
FROM customers c
LEFT JOIN orders o
ON c.customer_id = o.customer_id;
```

**Результат:**

| name           | city            | product | amount |
| -------------- | --------------- | ------- | ------ |
| Иван Петров    | Москва          | Ноутбук | 50000  |
| Иван Петров    | Москва          | Мышь    | 1000   |
| Мария Сидорова | Санкт-Петербург | Телефон | 30000  |
| Алексей Иванов | Казань          | NULL    | NULL   |
| Елена Смирнова | Москва          | NULL    | NULL   |

**Пример 2: Поиск клиентов без заказов**

```sql
SELECT
    c.name,
    c.city
FROM customers c
LEFT JOIN orders o
ON c.customer_id = o.customer_id
WHERE o.order_id IS NULL;
```

**Результат:**

| name           | city   |
| -------------- | ------ |
| Алексей Иванов | Казань |
| Елена Смирнова | Москва |

**Пример 3: LEFT JOIN с COALESCE**

```sql
SELECT
    c.name,
    COALESCE(SUM(o.amount), 0) AS total_spent
FROM customers c
LEFT JOIN orders o
ON c.customer_id = o.customer_id
GROUP BY c.customer_id, c.name;
```
