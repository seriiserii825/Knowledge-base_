## RIGHT JOIN (RIGHT OUTER JOIN)

RIGHT JOIN возвращает все строки из правой таблицы и совпадающие строки из левой таблицы.
Если совпадений нет, возвращаются NULL для столбцов левой таблицы.

### Синтаксис

```sql
SELECT columns
FROM table1
RIGHT JOIN table2
ON table1.column = table2.column;
```

### Примеры запросов

**Пример 1: Простой RIGHT JOIN**

```sql
SELECT
    c.name,
    o.product,
    o.amount
FROM customers c
RIGHT JOIN orders o
ON c.customer_id = o.customer_id;
```

**Результат:**

| name           | product    | amount |
| -------------- | ---------- | ------ |
| Иван Петров    | Ноутбук    | 50000  |
| Мария Сидорова | Телефон    | 30000  |
| Иван Петров    | Мышь       | 1000   |
| NULL           | Клавиатура | 3000   |

**Пример 2: Поиск заказов без клиентов (сироты)**

```sql
SELECT
    o.order_id,
    o.product,
    o.amount
FROM customers c
RIGHT JOIN orders o
ON c.customer_id = o.customer_id
WHERE c.customer_id IS NULL;
```

## FULL OUTER JOIN

FULL OUTER JOIN возвращает все строки из обеих таблиц. Если совпадение не найдено, NULL заполняет недостающие данные.

### Синтаксис

```sql
SELECT columns
FROM table1
FULL OUTER JOIN table2
ON table1.column = table2.column;
```

### Примеры запросов

**Пример 1: Простой FULL OUTER JOIN**

```sql
SELECT
    c.name,
    c.city,
    o.product,
    o.amount
FROM customers c
FULL OUTER JOIN orders o
ON c.customer_id = o.customer_id;
```

**Результат:**

| name           | city            | product    | amount |
| -------------- | --------------- | ---------- | ------ |
| Иван Петров    | Москва          | Ноутбук    | 50000  |
| Иван Петров    | Москва          | Мышь       | 1000   |
| Мария Сидорова | Санкт-Петербург | Телефон    | 30000  |
| Алексей Иванов | Казань          | NULL       | NULL   |
| Елена Смирнова | Москва          | NULL       | NULL   |
| NULL           | NULL            | Клавиатура | 3000   |

**Пример 2: Поиск несовпадающих записей**

```sql
SELECT
    c.customer_id AS cust_id,
    o.customer_id AS order_cust_id,
    c.name,
    o.product
FROM customers c
FULL OUTER JOIN orders o
ON c.customer_id = o.customer_id
WHERE c.customer_id IS NULL OR o.customer_id IS NULL;
```
