## Множественные JOIN

Можно объединять более двух таблиц в одном запросе.

### Пример данных

**Таблица: products**

| product_id | name    | category_id |
| ---------- | ------- | ----------- |
| 1          | Ноутбук | 1           |
| 2          | Телефон | 1           |
| 3          | Стол    | 2           |

**Таблица: categories**

| category_id | name        |
| ----------- | ----------- |
| 1           | Электроника |
| 2           | Мебель      |

**Таблица: order_items**

| item_id | order_id | product_id | quantity |
| ------- | -------- | ---------- | -------- |
| 1       | 101      | 1          | 2        |
| 2       | 101      | 2          | 1        |
| 3       | 102      | 3          | 1        |
