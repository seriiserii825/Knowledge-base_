## CROSS JOIN

CROSS JOIN возвращает декартово произведение двух таблиц (каждая строка первой таблицы соединяется с каждой строкой второй таблицы).

### Синтаксис

```sql
SELECT columns
FROM table1
CROSS JOIN table2;
```

### Пример данных

**Таблица: colors**

| color_id | color_name |
| -------- | ---------- |
| 1        | Красный    |
| 2        | Синий      |

**Таблица: sizes**

| size_id | size_name |
| ------- | --------- |
| 1       | S         |
| 2       | M         |
| 3       | L         |

### Примеры запросов

**Пример 1: Создание всех комбинаций**

```sql
SELECT
    c.color_name,
    s.size_name
FROM colors c
CROSS JOIN sizes s;
```

**Результат:**

| color_name | size_name |
| ---------- | --------- |
| Красный    | S         |
| Красный    | M         |
| Красный    | L         |
| Синий      | S         |
| Синий      | M         |
| Синий      | L         |

**Пример 2: CROSS JOIN с условиями**

```sql
SELECT
    c.color_name,
    s.size_name,
    CONCAT(c.color_name, ' - ', s.size_name) AS variant
FROM colors c
CROSS JOIN sizes s
WHERE s.size_name IN ('M', 'L');
```
