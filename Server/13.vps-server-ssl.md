domain seriiburduja.org left bar ssl

Создайте Origin Certificate

На этой же странице в Cloudflare:

Прокрутите вниз или перейдите в SSL/TLS → Origin Server (в левом меню)
Нажмите Create Certificate
Появится окно с настройками:
Private key type: оставьте RSA (2048)
Certificate Validity: можете выбрать 15 лет
Hostnames: должны быть seriiburduja.org и \*.seriiburduja.org
Нажмите Create
После этого вы увидите два поля:

Origin Certificate (верхнее поле)
Будет выглядеть примерно так:

-----BEGIN CERTIFICATE-----
MIIEpDCCA4ygAwIBAgIUAbc...
(много строк текста)
...xyz123
-----END CERTIFICATE-----
Private Key (нижнее поле)
Будет выглядеть примерно так:

-----BEGIN PRIVATE KEY-----
MIIEvQIBADANBgkqhki...
(много строк текста)
...abc456
-----END PRIVATE KEY-----
Скопируйте оба (полностью, со всеми строками -----BEGIN...----- и -----END...-----)

### 3. Установите сертификат на VPS

Подключитесь к VPS:

```bash
ssh root@72.62.93.126
```

Создайте директорию для сертификатов:

```bash
mkdir -p /etc/ssl/cloudflare
```

Создайте файл для сертификата:

````bash
nano /etc/ssl/cloudflare/seriiburduja.org.pem
    ```
Вставьте Origin Certificate из Cloudflare (весь текст, включая -----BEGIN CERTIFICATE----- и -----END CERTIFICATE-----)

Сохраните: Ctrl+O, Enter, Ctrl+X

Создайте файл для приватного ключа:


```bash
nano /etc/ssl/cloudflare/seriiburduja.org.key
````

Вставьте Private Key из Cloudflare

Сохраните: Ctrl+O, Enter, Ctrl+X

Установите правильные права:

```bash
chmod 600 /etc/ssl/cloudflare/seriiburduja.org.key
```

4. Обновите конфигурацию Nginx

```bash
nano /etc/nginx/sites-available/seriiburduja.org
```

Замените содержимое на:

````vim
server {
    listen 80;
    listen [::]:80;
    server_name seriiburduja.org www.seriiburduja.org;

# Редирект на HTTPS
    return 301 https://$server_name$request_uri;
}

server {
    listen 443 ssl http2;
    listen [::]:443 ssl http2;

    server_name seriiburduja.org www.seriiburduja.org;

# SSL сертификаты от Cloudflare
    ssl_certificate /etc/ssl/cloudflare/seriiburduja.org.pem;
    ssl_certificate_key /etc/ssl/cloudflare/seriiburduja.org.key;

# SSL настройки
    ssl_protocols TLSv1.2 TLSv1.3;
    ssl_ciphers HIGH:!aNULL:!MD5;
    ssl_prefer_server_ciphers on;

    root /var/www/seriiburduja.org;
    index index.html index.htm;

    location / {
        try_files $uri $uri/ =404;
    }
    ```
}
Сохраните: Ctrl+O, Enter, Ctrl+X

### 5. Проверьте и перезапустите Nginx


```bash
nginx -t
systemctl restart nginx
````
