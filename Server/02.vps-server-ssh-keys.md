### Generating SSH Keys
```bash
ssh-keygen -t ed25519 -C "seriiburduja@mail.com"
```
adding file name when prompted, e.g. /home/youruser/.ssh/id_ed25519_hostinger


### Add the key to ssh-agent

```bash
eval "$(ssh-agent -s)"
ssh-add ~/.ssh/id_ed25519_github
```

```bash
ssh-add -l
```

## copy key to vpsserver

```bash
ssh-copy-id -i ~/.ssh/id_ed25519_github.pub youruser@yourvpsserverip
```

### Testing the connection

```bash
ssh -i /home/serii/.ssh/id_ed25519_hostinger youruser@yourvpsserverip
```
