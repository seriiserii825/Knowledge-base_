### change port

```bash
sudo vim /etc/ssh/sshd_config
Port 2222  # указываем нужный порт
```

### 3. Проверьте синтаксис конфигурации

```bash
sudo sshd -t
```

### 5. Проверьте порт

```bash
sudo ss -tlnp | grep ssh
```

если порт не 2222, удаляем сокет

```bash
# Проверьте, есть ли socket unit
sudo systemctl status ssh.socket

# Если существует, отключите его
sudo systemctl stop ssh.socket
sudo systemctl disable ssh.socket

# Перезапустите SSH service
sudo systemctl restart ssh

# Проверьте порт
sudo ss -tlnp | grep ssh
```

### 6. проверяем доступ по порту в другом терминале, текущем не закрываем

```bash
ssh -p 2222 node@your_server_ip
```
