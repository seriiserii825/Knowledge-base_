### connect to vps with root
```bash
ssh root@your_server_ip
```
### 1. Создайте конфигурацию для вашего домена

Подключитесь к VPS и создайте файл конфигурации:


```bash
ssh root@72.62.93.126
```
Затем создайте конфигурационный файл:


```bash
nano /etc/nginx/sites-available/seriiburduja.org
```
Вставьте эту конфигурацию:


nginx

```vim
server {
    listen 80;
    listen [::]:80;

    server_name seriiburduja.org www.seriiburduja.org;

    root /var/www/seriiburduja.org;
    index index.html index.htm;

    location / {
        try_files $uri $uri/ =404;
    }
}
```
Сохраните файл: Ctrl+O, Enter, затем Ctrl+X

### 2. Создайте директорию для сайта


```bash
mkdir -p /var/www/seriiburduja.org
```
### 3. Создайте тестовую страницу


```bash
nano /var/www/seriiburduja.org/index.html
```
Вставьте:


```vim
<!DOCTYPE html>
<html>
<head>
<title>Welcome to seriiburduja.org</title>
</head>
<body>
<h1>Success! Your domain is working!</h1>
<p>seriiburduja.org is now connected to your VPS.</p>
</body>
</html>
```
Сохраните: Ctrl+O, Enter, Ctrl+X

### 4. Активируйте конфигурацию


```bash
ln -s /etc/nginx/sites-available/seriiburduja.org /etc/nginx/sites-enabled/
```
### 5. Проверьте конфигурацию и перезапустите Nginx


```bash
nginx -t
systemctl restart nginx
```
### 6. Проверка

Откройте в браузере: http://seriiburduja.org или http://www.seriiburduja.org

Вы должны увидеть тестовую страницу!
