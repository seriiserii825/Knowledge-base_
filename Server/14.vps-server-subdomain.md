### 1. Cloudflare DNS Configuration

First, let's add the DNS record in Cloudflare:

- Log in to your Cloudflare account
- Select your domain seriiburduja.org
- Go to DNS > Records
- Click Add record
- Configure:

```bash
Type: A
Name: nuxt-roman-timoshyuk
IPv4 address: Your VPS IP address
Proxy status: Proxied (orange cloud) - recommended
TTL: Auto
Click Save
```

### 2. VPS Server Configuration

Now let's set up the Nginx configuration. I'll create a configuration file for you:

Create Nginx config:

```bash
sudo nano /etc/nginx/sites-available/nuxt-roman-timoshyuk.seriiburduja.org
```

Add this configuration:

```vim
server {
    listen 80;
    listen [::]:80;
    server_name nuxt-roman-timoshyuk.seriiburduja.org;

# Redirect HTTP to HTTPS (after SSL is set up)
# return 301 https://$server_name$request_uri;

    location / {
        proxy_pass http://localhost:3000;  # Adjust port if your Nuxt app runs on different port
            proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection 'upgrade';
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
        proxy_cache_bypass $http_upgrade;
    }
}
```

Enable the site:

```bash
sudo ln -s /etc/nginx/sites-available/nuxt-roman-timoshyuk.seriiburduja.org /etc/nginx/sites-enabled/
```

Test and reload Nginx:

```bash
sudo nginx -t
sudo systemctl reload nginx
```

### 3. Set Up SSL with Let's Encrypt

```bash
sudo certbot --nginx -d nuxt-roman-timoshyuk.seriiburduja.org
```

### 4. Start Your Nuxt Application

Navigate to your project and start it:

```bash
cd /home/node/web/nuxt-roman-timoshyuk
npm install  # if dependencies aren't installed
npm run build  # build for production
npm run start  # or use PM2 for process management
```

Using PM2 (recommended):

```bash
pm2 start npm --name "nuxt-roman-timoshyuk" -- start
pm2 save
```
