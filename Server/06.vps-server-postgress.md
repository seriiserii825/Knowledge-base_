### install postregress

```bash
sudo apt update
sudo apt install postgresql-client -y

```

пароль находится в docker-composer.yml

```bash
  postgres:
    image: 'postgres:16'
    restart: always
    environment:
      POSTGRES_USER: '${DB_USERNAME}'
      POSTGRES_PASSWORD: '${DB_PASSWORD}'
      POSTGRES_DB: '${DB_NAME}'
```
