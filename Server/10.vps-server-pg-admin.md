
### после запуска если добавили .env нужно перезапустить контейнеры
```bash
docker compose down -v
docker compose up -d
```

### посмотреть с какими настройками залогинился
```bash
docker exec -it <pgadmin_container_name> env | grep PGADMIN
```

### restore backup on vps
```bash

docker exec -i nest-roman-timoshyuk-postgres-1 pg_restore -U postgres -d budget < ./backups/dump.backup
```
