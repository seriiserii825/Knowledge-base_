### 3️⃣ Создание нового пользователя (например node)

```bash
adduser node
```

Задай пароль, остальное можно пропустить.

Добавляем в sudo:

```bash
# Method 1: Add user to sudo group (more explicit)
usermod -aG sudo node

# Verify the user is in the sudo group
groups node
```

Проверка:

```bash
su - node
sudo whoami
```

Если вывод root — всё ок.
