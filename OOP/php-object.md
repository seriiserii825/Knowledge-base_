# Передача по ссылке
$a = new A();
$b = $a;
$a->name = 'Vasya';
$b->name = 'Petya';
var_dump($a->name); 'Petya'
var_dump($b->name); 'Petya'

Обе значения будут Petya, потом что объект не копируется, а передается по ссылке и если меняем $b, то меняется и $a;
$b = 10; то и $a тоже будет 10;

## Осторожно
$b = $a;
$a = 10;
$b - будет ссылаться на объект
$a = 10;

Но, если
$b = null;
То некого будет ссылаться на объект, и он останется в пространстве, и потом придет сборщик муссора и удалит. 

# Копировать объект
$b = clone $a;
теперь можем менять $b, и $a не будет менятся, потому что указатели введут на разные объекты
