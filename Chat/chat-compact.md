## `/compact` — сжатие контекста

Представь, что у Claude есть «рабочая память» — контекстное окно. Чем длиннее сессия, тем оно заполняется, и в какой-то момент Claude начинает «забывать», что было в начале разговора.

`/compact` решает это так: берёт всю историю разговора и сжимает её в плотное резюме, сохраняя самое важное. Разговор как бы «складывается» — детали уходят, суть остаётся.

---

### Разница с `/clear`

|                        | `/compact`                      | `/clear`                               |
| ---------------------- | ------------------------------- | -------------------------------------- |
| **Что делает**         | Сжимает историю в резюме        | Удаляет всё полностью                  |
| **Контекст**           | Частично сохраняется            | Теряется весь                          |
| **Когда использовать** | Долгая сессия, нужно продолжать | Новая задача, старый контекст не нужен |

---

### Примеры

```bash
/compact
# Claude сам решает, что важно сохранить

/compact retain the auth module and current test failures
# Говоришь Claude, что именно сохранить в резюме

/compact focus on the database schema decisions
# Полезно, если работал над несколькими вещами
# и хочешь, чтобы в резюме осталась только одна тема
```
