## `/rewind` — откат назад

`/rewind` — это твоя страховочная сеть. Особенно полезна, когда Claude пошёл по неверному пути — неправильно понял задачу или выбранный подход не работает.

Вместо того чтобы вручную откатывать файлы и объяснять, что пошло не так, просто вызови `/rewind` и начни эту часть разговора заново с более чётким промптом.

> Batsov

---

Нажми `Esc` дважды, чтобы войти в режим **rewind**. Появится меню с вариантами:

- **Restore code and conversation** — откатывает и файлы, и чат
- **Restore conversation** — перематывает чат, но сохраняет код
- **Restore code** — откатывает файлы, но сохраняет разговор
- **Summarize from here** — сворачивает всё с этой точки
- **Never mind** — отменяет и возвращает туда, где ты был

> DEV Community

### Пример использования

```bash
# Claude написал неудачный рефакторинг — нажимаешь Esc+Esc,
# выбираешь "Restore code" → код откатился, разговор остался
```
