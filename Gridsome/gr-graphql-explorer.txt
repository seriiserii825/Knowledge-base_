The 1-minute install Install the vscode-graphiql-explorer-extension. 
In your repository root, while your development server is running, generate a GraphQL schema for the plugin with following command: npx get-graphql-schema http://localhost:8080/___graphql > schema.graphql (the first part of the URL should be your development server URL).
You might want to add schema.graphql to gitignore as well. 
Profit!11 You can now use the GraphiQL explorer inside your VS Code. 
Run Explore schema with GraphiQL command to see it in action.
