# Работа с `tar` для больших папок с изображениями

## Установка `pv`

### Arch Linux

```bash
sudo pacman -S pv
```

---

## Создание архива с прогрессом

```bash
tar -cf - images/ | pv > images.tar
```

Во время создания архива будет отображаться прогресс:

```text
1.2GiB 0:00:15 [82.3MiB/s] [=========>      ]
```

Показывается:

- общий объём обработанных данных;
- прошедшее время;
- текущая скорость;
- индикатор выполнения.

> Для больших папок с изображениями (`jpg`, `jpeg`, `png`, `webp`) это один из самых удобных способов создания архива.

---

## Проверить содержимое архива

```bash
tar -tf images.tar
```

Выведет список всех файлов внутри архива без распаковки.

---

## Распаковать архив

### В текущую директорию

```bash
tar -xf images.tar
```

---

### В другую директорию

```bash
mkdir extracted

tar -xf images.tar -C extracted
```

---

## Посмотреть размер архива

```bash
ls -lh images.tar
```

# Распаковка `tar` с прогрессом

## В текущую директорию

```bash
pv images.tar | tar -xf -
```

---

## В указанную директорию

```bash
mkdir extracted

pv images.tar | tar -xf - -C extracted
```

Во время распаковки будет отображаться прогресс:

```text
1.2GiB 0:00:15 [82.3MiB/s] [=========>      ]
```

Показывается:

- объём обработанных данных;
- прошедшее время;
- скорость распаковки;
- индикатор выполнения.

> Для больших архивов (несколько гигабайт) это самый удобный способ отслеживать процесс распаковки.
