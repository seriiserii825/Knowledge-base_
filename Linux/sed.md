## Замена grep
```php
sed -n -e '/color/p' sed.scss
-n - выводить только строки с совпадением
-e - команда
p - строки которые соответсвуют команде
``` 

## Вывод строк
```php
sed 10q file
q - вывести строки и закончить операцию
```

```php
sed -n -e '5,10p' file
, - диапозон
```

```
sed 5d file
5d - все строки кроме 5
```

```
sed 5!d file
только 5 строку
```

## Редактирование строк
sed -i s/400/576/g sed.scss

## Удаление строк
```
// покажет только вывод
sed "/\/\//d" sed.scss
// удалить в файле
sed -i "/\/\//d" sed.scss
```

## Несколько фильтров
```
sed -e /^$/d -e /^\*/d sed.scss
```
