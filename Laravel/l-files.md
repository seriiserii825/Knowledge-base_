app/
	Console/ - доступ к api фреймворка по протоколу click
	Events/ - классы событий
	Exceptions/ - классы, которые используются для работы с исключениями
	Http/
	- Controllers/ - контроллеры
	- Middleware/ - посредники
	- Requests/ - классы запроса
	- Kernel ядро запроса, который задается через протокол http
	- Routes - маршруты, которые формируют функционал будущего приложения
	Jobs/ - классы очередей для работы с задачами
	Listeners/ - обработчики событий, код которых срабатывает при выполнения того или иного события
	Policies/ - классы политики безопасности, прав пользователя для выполнения некого действия
	Providers/ - сервис провайдеры приложения
bootstrap/ - файлы, которые обеспечивают загрузку фреймворка
config/ - конфигурационные файлы приложения
database/ - Миграции и посев данных
public/ - глобальная точка входа во фреймворк Laravel.
resources/ - файлы views, локализации и less, sass приложений
storage/ - скомпилируемые blade шаблоны, файлы сесий, кэша и другие файлы.
tests/ - проведения unit тестов
vendor/ - файлы композера
