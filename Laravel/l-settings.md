#Настройки
##.env
**APP_ENV**=local

*Все параметры из этого файла относятся к среде разработки локал.*

**APP_DEBUG** - для локальной разработки можно включить

**APP_KEY** - строка в 32 символа для шифорвания данных в фреймворке

В файле **.env.example** - хранится копия наших настроек, чтобы другой человек мог видеть наши добавленные настройки

Когда проект запускается, все данные из файла .env попадает в масив $_ENV[]; 

В файле app.php задаются значения по-умолчанию для некоторых настроек из файла .env.

В файле app.php можно поменять url и uct. 

Если в файле настройки, например, database.php есть настройка 'default' => env('DB_CONNECTION', 'mysql'). В файле .env такой настройки нету, потому мы ее можем записать сами и выбрать тип базы данных с которой будем работать.
