# Init

```bash
uv init
```

## Sync for existing project

```bash
uv lock
uv sync
```

## add package

```bash
uv add <project_name>
```

## remove package

```bash
uv remove <project_name>
```

## run package

```bash
uv run <project_name>
```

## by default another version of python

```bash
uv python pin 3.11
```
