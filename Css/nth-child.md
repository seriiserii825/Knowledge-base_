/_ =========================================
КАК РАБОТАЕТ nth-child / nth-of-type
========================================= _/

/_
3n = каждый третий элемент
3n+1 = 1,4,7,10...
3n+2 = 2,5,8,11...
3n+3 = 3,6,9,12...
n+6 = начиная с 6-го элемента
-n+5 = только первые 5 элементов
_/

/_ -----------------------------------------
Каждый 3-й начиная со 2-го
----------------------------------------- _/

li:nth-child(3n+2) {
color: green;
}

/_
Выберет:
2,5,8,11...
_/

/_ -----------------------------------------
Все кроме первых пяти
----------------------------------------- _/

li:nth-child(n+6) {
color: green;
}

/_
6,7,8,9...
_/

/_ -----------------------------------------
Только первые пять
----------------------------------------- _/

li:nth-child(-n+5) {
color: green;
}

/_
1,2,3,4,5
_/

/_ -----------------------------------------
Предпоследний элемент
----------------------------------------- _/

li:nth-last-child(2) {
color: green;
}

/_
Если элементов 10:
выберет 9-й
_/

/_ =========================================
ТВОЙ ПРИМЕР ПО 3 ЭЛЕМЕНТАМ
========================================= _/

/\*
Повторяется группа из 3 блоков:

1-й -> большой
2-й -> маленький
3-й -> маленький

Потом снова:
4-й -> большой
5-й -> маленький
6-й -> маленький
\*/

.item {
&:nth-of-type(3n+1) {
width: 40%;
background: green;
}

&:nth-of-type(3n+2) {
width: 20%;
}

&:nth-of-type(3n+3) {
width: 20%;
}
}

/_
1,4,7,10... = большие
2,5,8,11... = маленькие
3,6,9,12... = маленькие
_/
