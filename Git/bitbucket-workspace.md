Как создать новый workspace:

Открой:
https://admin.atlassian.com

Войди в свой Atlassian account

Слева:
Apps / Atlassian apps

Нажми:
Add app

Выбери:
Bitbucket

Введи:

Workspace name

workspace ID создастся автоматически (или можно изменить, если доступно)

Нажми:
Agree and add

После этого новый workspace появится в Bitbucket.
