# zf
Then you can use zf operator to mark a piece text as foldable. zf can be used with text objects (i.e. zfap marks the entire paragraph) as well as
in visual mode (for example, hit V, then hit j multiple times to select how many lines you want, then zf to make them foldable).

# zM zR
There is also zM and zR to close / open all the folds in the file,
no matter how deep. That's super useful with big files when you need to get an overview of the whole file.

# za
And of course, let's not forget about za, which works like a toggle.
When the cursor is on an open fold, it will close it and the other way around.
I'm using it so often that I have it mapped to Space:

# zo 
open

# zc
close
