# fold levels
```
zj moves the cursor to the next fold.
[z moves the cursor to the start of the current fold.
]z moves the cursor to the end of the current fold.
zk moves the cursor to the previous fold.
zo opens a fold at the cursor.
zO opens all folds at the cursor.
zc closes a fold under cursor. 
zC closes all folds untill parrent when cursor is in a child
zM closes all open folds.
zR open all folds
zm increases the foldlevel by one.
zr decreases the foldlevel by one.
zN close and remember folding
zn open and remember folding
zi toggle folding
```
