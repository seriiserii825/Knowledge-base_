## ✔️ Добавить одну пустую строку после строки 12

Из Normal mode:

```vim
:12put _
```

Это вставит одну пустую строку под 12-й строкой.

## ✔️ Добавить две пустые строки (например, между 12 и 14)

```vim
:12put _ | 12put _
```

или более красиво:

```vim
:12put =['','']
```

## ✔️ Добавить N пустых строк

Например, 5 пустых:

```vim
:12put =repeat([''], 5)
```
