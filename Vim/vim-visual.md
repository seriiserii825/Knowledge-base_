## columns

```vim
Chapter            Page
Normal mode          15

```

```vim
<C-v>3j - select 3 lines down in visual block mode
x... - delete 3 lines for current column
gv - reselect the visual block
r|Chapter - replace with vertical line
yyp - copy and paste the line
Vr- - replace line with a -
```
