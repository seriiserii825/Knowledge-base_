# targets.vim Cheat Sheet

## Аргументы

```js
foo(first, second, third, fourth, fifth);
```

| Команда | Результат              |
| ------- | ---------------------- |
| `cia`   | текущий аргумент       |
| `cina`  | следующий аргумент     |
| `cila`  | предыдущий аргумент    |
| `c2ina` | 2 следующих аргумента  |
| `c3ila` | 3 предыдущих аргумента |

---

## Разделители

```txt
apple, banana, orange, kiwi
```

| Команда | Результат             |
| ------- | --------------------- |
| `ci,`   | текущий элемент       |
| `cin,`  | следующий элемент     |
| `cil,`  | предыдущий элемент    |
| `c2in,` | 2 следующих элемента  |
| `c3il,` | 3 предыдущих элемента |

Работает также для:

```txt
. ; : + - = / | &
```

---

## Любые скобки

```js
foo(bar)
foo[bar]
foo{bar}
```

| Команда | Результат           |
| ------- | ------------------- |
| `cib`   | внутри любых скобок |
| `cab`   | вместе со скобками  |

`b = (), [], {}`

---

## Любые кавычки

```js
"hello";
"hello"`hello`;
```

| Команда | Результат            |
| ------- | -------------------- |
| `ciq`   | внутри любых кавычек |
| `caq`   | вместе с кавычками   |

`q = ", ', \``

---

## Следующий объект

Курсор может быть где угодно:

```js
foo(bar);
```

| Команда | Результат      |
| ------- | -------------- |
| `cin(`  | следующие `()` |
| `cin[`  | следующие `[]` |
| `cin{`  | следующие `{}` |
| `cin"`  | следующие `""` |
| `cin'`  | следующие `''` |

---

## Предыдущий объект

| Команда | Результат       |
| ------- | --------------- |
| `cil(`  | предыдущие `()` |
| `cil[`  | предыдущие `[]` |
| `cil{`  | предыдущие `{}` |
| `cil"`  | предыдущие `""` |
| `cil'`  | предыдущие `''` |

---
