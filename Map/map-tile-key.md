Инструкция по получению ключа MapTiler:

1. Зайди на cloud.maptiler.com и зарегистрируйся (бесплатно, через email или Google/GitHub).
2. После входа попадёшь в Dashboard — там сразу будет создан Default key (или создай новый через раздел "API keys" → "Create a new key").
3. Скопируй ключ — это строка вида AbCdEf12345....
4. URL тайлов будет вида:
   https://api.maptiler.com/maps/streets-v2/{z}/{x}/{y}.png?key=ТВОЙ_КЛЮЧ
5. (вместо streets-v2 можно подставить другие стили: basic-v2, outdoor-v2, toner-v2, dataviz и т.д. — полный список в Dashboard → "Maps").
6. В Dashboard можно ограничить ключ по доменам (allowed URLs), чтобы его не использовали с других сайтов.
7. Бесплатный тариф даёт лимит запросов в месяц — в Dashboard есть счётчик использования, можно следить, не приближаешься ли к лимиту.
